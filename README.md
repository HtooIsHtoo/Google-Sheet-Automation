# 🚀 Daily Revenue Report Automation

Stop checking spreadsheets manually.

This tool automatically sends a clean daily revenue report from your Google Sheet to your email.

## ✨ Features
- Daily revenue summary
- Growth vs yesterday
- Product breakdown
- Automated email reports
- Runs inside your Google account

## ⚡ Setup (5 minutes)

1. Open Google Sheets
2. Go to Extensions → Apps Script
3. Paste the provided scripts
4. Run setup
5. Receive daily reports automatically
